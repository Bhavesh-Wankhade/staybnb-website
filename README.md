# Staybnb Website

Staybnb is a responsive landing page for a travel and stay booking platform, designed using **HTML, CSS, and JavaScript**. The website allows users to explore popular and trending destinations, search for stays, and view traveler stories.

## Features

- **Responsive Design**: Works perfectly on desktops, tablets, and mobile devices.
- **Search Functionality**: Users can search by location, check-in/check-out dates, and number of guests.
- **Exclusives Section**: Displays popular travel destinations with pricing.
- **Trending Places**: Highlights trending locations for travelers.
- **Traveler Stories**: Showcases experiences shared by travelers.
- **Call to Action**: Encourages users to share extra space and earn money.
- **About Section**: Information about Staybnb.
- **Social Media Footer**: Links to major social platforms.

## Technologies Used

- **HTML5** for the structure of the webpage.
- **CSS3** for styling and responsive layout.
- **JavaScript** for interactive elements, such as the mobile navbar toggle.
- **Font Awesome** for icons.
- **Images** for banners, destinations, and traveler stories.
